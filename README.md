# SailboatTrackerApp

**Icon**

![Image of 1](https://images2.imgbox.com/75/87/8MRFDtyg_o.jpg)

This application helps to track the boats progress during the race. It takes data from the official site and displays it accordingly. The main purpose is to help followers to get the race information quicker and simpler. 

This App:
- contains no adds
- asks no passwords or sensible data of any kind
If anything of the list above is not true during the app usage, please let me know.

- minimum Android version: API level 19 (4.4 KitKat)
- Google Play Services
- Internet Access

**Tested on**
- API level 25 (7.1 Nougat), ASUS Zenfone

**Additional info**

During tests the traffic data amount was measured and size of it was around 1Mb per boats position request.
